## 🎯 간소화 목표

### 기존 문제점
- **에셋 과다**: 총 74개 파일 (이미지 47개, 영상 5개, 오디오 20개, 폰트 2개)
- **제작 부담**: 4주 이상 소요, 외주 비용 $1,000 이상
- **용량 문제**: 총 ~50MB, 모바일 로딩 부담

### 해결 방안
- **에셋 50% 축소**: 74개 → 37개
- **제작 기간 단축**: 4주 → 2주
- **용량 최적화**: 50MB → 20MB 이하

## 📋 간소화 전략

### 1. 배경 이미지 축소 (8개 → 3개)
**유지:**
- `countryside.webp` - 시골 (Chapter 1 + Ending A)
- `seoul_city.webp` - 서울 거리 (Chapter 2, 3)
- `seoul_luxury.webp` - 부잣집 주방 (Chapter 3, 4)

**제거:**

- ~~kitchen_day/night~~ → seoul_luxury로 통합
- ~~rainy_alley~~ → 색상 필터로 처리
- ~~pipe_inside~~ → 검은 배경 + 효과

### 2. 캐릭터 일러스트 축소 (12개 → 4개)
**시골이 (2개):**
- `countryside_neutral.webp` - 기본 (중립/행복/호기심 겸용)
- `countryside_worried.webp` - 불안/공포/성찰 겸용

**서울이 (2개):**
- `seoul_neutral.webp` - 기본 (자신감/자랑 겸용)
- `seoul_serious.webp` - 짜증/진지 겸용

**제거 전략:**
- 표정 변화는 CSS 필터 + 텍스트로 보완
- 8개 표정 → 2개로 통합 (긍정/부정)

### 3. 엔딩 일러스트 축소 (4개 → 2개)
**유지:**
- `ending_good.webp` - 긍정 엔딩 (A, B, C 겸용)
- `ending_bad.webp` - 부정 엔딩 (Hidden)

**제거 전략:**
- 각 엔딩마다 다른 텍스트로 차별화
- 색상 필터로 분위기 변경

### 4. 영상 축소 (5개 → 2개, 각 5초)
**유지:**

**제거:**

### 6. 수집 아이템 제거 (8개 → 0개)
**제거 이유:**
- 게임플레이에 필수 아님
- 갤러리 복잡도 증가
